# 재귀함수이론

## 재귀함수란

- 자기 자신을 호출하는 함수

- e.g.

  ```class Main {
        public void myFunction(int n) {
            // do something
            myFunction(n - 1);
            // do something
        }
     }
  ```

- 왜 사용할까?
  - 1. 하나의 **커다란 문제**를 **작은 문제**로 나누어 해결
  - 2. 문제를 귀납적으로 생각하기 위해서
       {i}번째 답을 구하기 위해 (i-1), (i-2)번째 결과 활용
